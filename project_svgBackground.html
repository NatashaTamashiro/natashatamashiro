SVG Background

Como aplicar/customizar de forma rápida um lindo background.

Personalizar a aparência de um site, muitas vezes não é uma simples tarefa. Definir uma cor ou textura que se encaixe, que seja harmonioza com as fontes, imagens e tudo mais.

Então, aqui vai uma dica muito útil. ~Que particularmente eu estou usando, rs

Primeiro, O que é um SVG?
Ele é um arquivo que permite imagens em alta resolução, com um tamanho de arquivo próximo a 5kb e são suportados por todos os navegadores modernos.

Segundo, Links:

  • https://www.svgbackgrounds.com/ ~Estou usando deste site!
  • https://www.heropatterns.com/ ~Mas este também é muito legal!

Por último, o Código!
Para aplicar, é bem simples:

  1. Clique no svg desejado e copie o código que irá aparecer.
    ( Note que você pode edita-lo. Cor, transparência entre outras opções )


  2. Cole o código css na tag que deseja inserir o background.
    ( Nesse caso, usei no Body )


  3. Cole e veja o resultado!
